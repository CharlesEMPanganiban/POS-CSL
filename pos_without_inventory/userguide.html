User Guide

1. Introduction

This system is a Point of Sale (POS) system that allows users to manage products, place orders, track sales, monitor inventory, and log employee activity.

2. User Roles

Admin:

  • Manage products (add, edit, delete)
  • View sales analytics
  • Access employee logs
  • Manage inventory

Employee:

  • Browse menu and place orders
  • View cart and complete transactions
  • Automatic time-in and time-out logging

3. Account Registration & Login

1. Create an account using a username and password.

2. Login using your credentials.

3. Admin account is automatically created:

Username: admin

Password: ********

4. Dashboard

After login, users will see the dashboard with the following options:

  • Menu – View and order products
  • Sales Report – View analytics and receipts
  • Add Product (Admin only)
  • Employee Log (Admin only)

5. Menu & Ordering

  • Select products from the menu
  • Click "+" to add items to cart
  • Adjust quantity or remove items
  • Click Order to confirm

Order Flow:

  • Orders go to Pending
  • Click ✔ to mark as Completed

6. Sales Analytics

  • Today – Shows the latest 20 receipts only
  • Weekly / Monthly / Yearly – Filters recent sales
  • Archive – Shows all older receipts (full history)

Note: Older receipts are automatically moved to Archive when sales exceed 20 entries.

8. Employee Log

  • Login = automatic Time-In
  • Logout = automatic Time-Out
  • Admin can view all employee logs

9. Data Storage

All data is stored locally in the browser using localStorage. This includes:

  • Users
  • Sales
  • Employee logs
